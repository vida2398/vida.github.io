Vida

소개 기술 스택 프로젝트 경험 교육

소개

컴퓨터공학을 전공하며 소프트웨어 개발과 시스템 프로그래밍에 관심이 있는 학생입니다.

기술 스택

Front-End: React, HTML/CSS

Back-End: Java, Spring, REST API

System: Linux, C, Raspberry Pi, Arduino, TCP/IP

AR/VR: Unity, C#

DB: MySQL

프로젝트

영어 대본 파트너 로봇

  • Raspberry Pi 기반 음성 시스템 개발
  • STT / TTS 기능 구현
  • 발음 평가 기능 실험

데이터 분석 프로젝트

  • Python + Streamlit 활용
  • 데이터 시각화 및 발표

경험

  • 요구사항 분석 및 기능 정의
  • 데이터베이스 모델링 (ERD)
  • TCP/IP 소켓 프로그래밍 기반 통신 구현

교육

컴퓨터공학 전공 (재학 중)
